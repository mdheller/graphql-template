# GraphQL Template

Quick start for React/GraphQL projects.

to test locally:

```
cd sub/data
sls offline start
```

this will run the data on port 9000 (e.g., http://localhost:9000)

then, in another terminal window:

```
cd sub/app
yarn start
```

this will run the app on port 3000 (e.g., http://localhost:3000)
